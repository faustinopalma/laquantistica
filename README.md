# Esperimenti fondamentali della Meccanica Quantistica

Edizione digitale della tesi di laurea di **Faustino Palma** (1999): un percorso
didattico che introduce la Meccanica Quantistica a partire dagli esperimenti
fondamentali e che **deduce** l'equazione di Schrödinger invece di postularla.

> Lavoro presentato all'85° Congresso Nazionale della Società Italiana di Fisica
> (SIF), Pavia, settembre 1999. Laboratorio didattico LAFIDIN — Università di
> Napoli «Federico II».

🌐 **Edizione web (IT/EN):** [laquantistica.com](https://laquantistica.com)

---

## Cos'è

La tesi esegue nove esperimenti storici della fisica atomica e, dalla loro
analisi, costruisce un'introduzione completa alla teoria quantistica, trattata
fin dall'inizio con le notazioni di Dirac. La derivazione dell'equazione di
Schrödinger non richiede la Meccanica Analitica: si appoggia solo sulla
meccanica di Newton e sull'elettromagnetismo.

Le nove «schede»:

1. Esperimento di Stern-Gerlach
2. Esperimenti di Stern-Gerlach in cascata
3. Esperimenti con gli Elettroni
4. Diffrazione degli Elettroni — *deduzione dell'equazione di Schrödinger*
5. Esperimento di Rutherford
6. Ulteriori sviluppi della Teoria
7. Esperimento di Franck-Hertz
8. Effetto Fotoelettrico
9. Spettri atomici di emissione
